## `truthy` value and `falsy` value:
### `falsy` values:
- `false`
- `0`
- `-0`
- `BingInt 0n`
- `{}`
- `null`
- `undefine`
- `NaN`

### `truthy` values:
- `true`
- `1`
- `"0"` => zero in string is a truthy value
- `flase`
- `" "` => space in string is a truthy value
- empty array (`[]`)
- empty object (`{}`)
- empty function (`function(){}`)
- `false == 0`
- `flase == ''`
- `0 == ''`
